Deriving the type of linked list algebraically Oscar Puente (1 min. read)

Posted on 2022-03-16

I have been studying Category Theory lately, and I came across this beautiful proof showing how to derive the type of linked list algebraically.

Let $T$ be some type, e.g. integer. Suppose we want to define a type $L(T)$ for a list of $T$. Note that, $$L(T) = \emptyset \ | \ T \ | \ T^2 \ | \ T^3 \ | \ ... $$

That is, $L(T)$ is either a container with no elements, or a container with one element, or a container with two elements, and so on. This can be written algebraically as $$L(T) = 1 + T + T^2 + T^3 + ...$$

Which is equivalent to, $$L(T) = \frac{1}{1 - T}$$

Transforming this expression, we get $$L(T) (1-T) = 1$$ $$L(T) = 1 + T \times L(T)$$

We can write this back in a programmatic notation as, $$L(T) = \emptyset \ | \ (T, L(T))$$

Meaning that $L(T)$ is either an empty container, or a pair containing an element of type $T$ in the $car$ and an element of type $L(T)$ in the $cdr$. This is just the definition of linked list we are familiar with!
